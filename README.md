# 💊 MediMet

A Medicine Reminder web application built using **Flask**, **MySQL**, **HTML**, **CSS**, and **JavaScript**. It helps users manage their daily medications with secure authentication and an easy-to-use dashboard.

## ✨ Features

- User Registration & Login
- Add, Edit & Delete Medicines
- Mark Medicines as Taken
- Search Medicines
- Medicine Dashboard
- MySQL Database Integration

## 🛠️ Tech Stack

- Python (Flask)
- MySQL
- HTML
- CSS
- JavaScript

## ▶️ Run the Project

```bash
pip install -r requirements.txt
python app.py
```

Open:

```
[http://127.0.0.1:5000](url)
```
